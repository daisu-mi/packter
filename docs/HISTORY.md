# プロジェクト系譜

PACKTER のビジュアルデザインの源流は、NICTER の 3D トラフィック可視化
「CUBE」にある。CUBE は門林雄基先生のアイデアを基に、衛藤将史氏のもとで
横河電機が開発したものである。PACKTER は同じ研究室に属していた宮本大輔が
開発した——つまり CUBE と PACKTER は同一研究室のアイデアの系譜上にあり、
PACKTER は「CUBE のライト版」として安心して捉えてよい（原作者談、2026-06）。

## 旧バージョンの歩み

- 2008-08 Packter 1.0 初版
- 2010-06 2.0–2.2: PACKTERMSG / SOUND / VOICE / SKYDOMETEXTURE 等の演出コマンド追加
- 2011-07 2.4: Viewer（XNA 3.1 + WinForms）
- 2012-08 2.5: Agent が sFlow / NetFlow 対応
- 2017-03 VR版エンジニアリングサンプル（Unity / HTC Vive）
- 2026-06 3.0 開発開始: Rustブローカー + Webビューア（本リポジトリ）
